# dotfiles

These are config files to set up a system the way I like it. It include config files for zsh, brew, gem, git and theme terminal.

<p align="center">
	<img src="screenshots/terminal.png" alt="Terminal" title="Terminal">
</p>

## Setup
```terminal
$ sh setup.sh
```
